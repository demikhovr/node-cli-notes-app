# Node.js command line note-taking app

## Install
```
npm i
```

## Usage

To add a new note:
```
node app add --title="<note title>" --body="<note body>"
```

To list all notes:
```
node app list
```

To read a note by title:
```
node app read --title="<note title>"
```

To remove a note:
```
node app remove --title="<note title>"
```
